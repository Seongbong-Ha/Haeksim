# Haeksim

## 📖 프로젝트 개요

AI 기반의 수능 비문학 독해 학습 플랫폼으로, 개인화된 지문 생성과 AI 튜터를 통해 효과적인 학습을 지원합니다. 수능 수준의 비문학 지문을 자동 생성하고, 학습자의 문제 풀이와 요약 작성을 실시간으로 분석하여 맞춤형 피드백을 제공합니다.

### 🎯 핵심 기능

- **AI 기반 지문 생성**: 수능 수준의 비문학 지문을 주제별, 난이도별로 자동 생성
- **문제 풀이 분석**: 선지별 근거 분석과 논리적 사고력 평가
- **요약 작성 평가**: 학습자가 작성한 요약문의 논리 구조와 핵심 파악도 분석
- **소크라테스식 AI 튜터**: 대화형 질의응답을 통한 능동적 학습 유도
- **개인화 학습 분석**: 학습 패턴과 취약점 분석을 통한 맞춤형 학습 경로 제안

### 🛠 기술 스택

#### 백엔드 & AI 모델
- **FastAPI**: 고성능 웹 API 프레임워크
- **OpenAI API**: GPT 모델을 활용한 지문 생성 및 분석
- **SQLAlchemy**: ORM을 통한 데이터베이스 관리
- **RAG (Retrieval Augmented Generation)**: 벡터 검색 기반 문서 유사도 분석
- **PostgreSQL**: 사용자 데이터 및 학습 기록 저장

#### 프론트엔드
- **React**: 사용자 인터페이스 구축
- **React Router**: SPA 라우팅
- **Axios**: HTTP 클라이언트
- **CSS3**: 반응형 디자인

## 🔧 설치 및 실행

### 사전 요구사항
- Node.js 14+
- Python 3.8+
- PostgreSQL
- OpenAI API Key

### 백엔드 실행
```bash
cd suneung-nonfiction
pip install -r requirements.txt
uvicorn app:app --reload --host 0.0.0.0 --port 8000
```

### 프론트엔드 실행
```bash
cd front
npm install
npm start
```

## 📚 주요 페이지

1. **홈페이지** (`/`) - 온보딩 및 서비스 소개
2. **로그인/회원가입** (`/page1`, `/sign-up`) - 사용자 인증

6. **퀴즈 결과** (`/quiz-results`) - 풀이 분석 및 피드백
7. **요약 연습** (`/summary-practice`) - 요약 작성 및 평가
8. **학습 분석** (`/learning-analysis`) - 종합 학습 리포트

## 🤖 AI 기능

### 지문 생성
- **RAG 기반 유사 문서 검색**: 기존 학습 데이터에서 유사한 주제의 문서 탐색
- **LLM 기반 지문 재작성**: 수능 출제 기준에 맞는 새로운 지문 생성
- **난이도 조절**: 어휘 수준, 문장 복잡도, 논리 구조 복잡도 조정

### 학습 분석
- **문제 풀이 분석**: 선지별 근거의 논리성과 정확성 평가
- **요약 품질 평가**: 논리성, 완성도, 핵심 파악도 점수화
- **학습 패턴 분석**: 개인별 강점과 약점 영역 식별

### AI 튜터
- **소크라테스식 대화**: 즉답 대신 유도 질문을 통한 사고력 향상
- **맥락 기반 대화**: 현재 풀이 중인 문제와 연관된 질의응답
- **학습 코칭**: 학습 방법과 전략에 대한 개인화된 조언

## 📊 데이터베이스 스키마

### 주요 테이블
- **Users**: 사용자 정보 및 인증
- **Items**: 생성된 지문과 문제
- **Choices**: 문제의 선지 정보
- **Submissions**: 학습자의 답안 제출 기록
- **Summaries**: 요약 작성 및 평가 기록
